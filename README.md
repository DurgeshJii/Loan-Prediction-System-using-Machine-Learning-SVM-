# Loan-Prediction-System-using-Machine-Learning-SVM-
This project implements a Loan Approval Prediction System using Support Vector Machine (SVM) to determine whether a loan application should be approved or rejected based on applicant details.

The workflow includes data preprocessing, feature engineering, exploratory data analysis, model training, evaluation, and a real-time prediction system.
